Polar

In this project in which we were to use 3D animation or virtual reality to create a web app. We decided to create an arctic polar web project in which visitors to the site get information on endagered animals in the arctic region and they get to donate to help the cause.

I used THREE JS to program the site and create a sort of a spherical 3D surrounding area where you can view the kind of animals from what you choose on the front page.
